جواب‌های Claude حرف ندارند، فقط مشکل اینجاست که فارسی‌اش چپ‌چین و به‌هم‌ریخته در می‌آید. این افزونه با یک کلیک حلش می‌کند و توی کار بقیه‌ی چیزها هم اصلا دخالت نمی‌کند.

| 😖 قبلش | 😍 بعدش |
| --- | --- |
| جواب‌های فارسی چپ‌چین و درهم می‌آیند | هر جواب تروتمیز راست‌چین می‌شود، با یک فونت خوش‌دست |
| موقع تایپ، نشانگر و حروف بالا و پایین می‌پرند | همین که شروع کنی به نوشتن، کادر تایپ راست‌چین می‌شود |
| کد و انگلیسی قاطی متن، شلخته به نظر می‌رسند | کد سر جای خودش می‌ماند، انگلیسی هم همین‌طور |

## ✨ چرا عاشقش می‌شوی

- 🪄 **خودکار** — جهت متن را پاراگراف به پاراگراف خودش تشخیص می‌دهد؛ نه دکمه‌ای، نه تنظیماتی.
- 🌍 **زبان خودت** — فارسی را بی‌نقص می‌چیند و با عربی، اردو، کردی و باقی خط‌های راست‌به‌چپ هم راه می‌آید.
- 🔤 **تایپوگرافی تمیز** — فونت وزیرمتن همراهش می‌آید، با اندازه و فاصله‌ای که خودت تنظیم می‌کنی.
- ⌨️ **کادر تایپ هوشمند** — تا دست به نوشتن می‌بری، خودش راست‌چین می‌شود.
- 🧩 **کاری به کد ندارد** — بلاک‌های کد، diff و ترمینال چپ‌به‌راست و دست‌نخورده می‌مانند.
- 🎛️ **کنترل دستی با یک کلیک** — روی هر پیام یک کلید کوچک هست، هر وقت خواستی خودت جهتش را عوض کن.
- 🧠 **فقط Claude نه** — روی ChatGPT و Gemini هم کار می‌کند.
- 🔒 **کاملا روی دستگاه خودت** — صددرصد لوکال؛ نه سروری، نه ردیابی‌ای.

## 🚀 نصب

هر بخشی را که لازم داری بردار؛ هرکدام یک بسته‌ی مستقل و کامل است.

**مرورگر — Chrome / Edge / Brave**

1. پوشه‌ی [`browser-extension`](browser-extension) را دانلود کن.
2. آدرس `chrome://extensions` را باز کن و **Developer mode** را روشن کن.
3. روی **Load unpacked** بزن و همان پوشه را انتخاب کن. تمام. ✅

**مرورگر — Firefox**

1. آدرس `about:debugging` را باز کن و روی **Load Temporary Add-on** بزن.
2. فایل `browser-extension/manifest.json` را انتخاب کن.

**VS Code — Claude Code**

از **Marketplace**: عبارتِ **«RTL for Claude»** را جستجو کن (یا
[صفحه‌اش](https://marketplace.visualstudio.com/items?itemName=mahdigh99.rtl-for-claude)
را باز کن)، روی **Install** بزن، بعد **Developer: Reload Window** را اجرا کن.

یا با فایلِ **`.vsix`**: فایلِ `rtl-for-claude-vscode-*.vsix` را از
[Releases](https://github.com/mahdigh99/rtl-for-claude/releases) دانلود کن و توی VS Code
**Extensions ← `…` ← Install from VSIX…** را بزن.

بدونِ نصب: `bash vscode-extension/apply-rtl.sh`. راهنمای کامل:
[vscode-extension/README.md](vscode-extension/README.md).

## 🎛️ به سلیقه‌ی خودت

روی آیکن نوار ابزار بزن و همه‌چیز را همان‌طور که دوست داری تنظیم کن: حساسیت تشخیص، فونت، اندازه‌ی متن، فاصله‌ی خط‌ها، کلید جهت روی هر پیام، و این‌که افزونه روی کدام سایت‌ها فعال باشد.
